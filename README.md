# **혼자 공부하는 파이썬**  

## **개요**  
이 레포지토리는 **2023년 1월 5일**부터 **1월 26일**까지 **혼자 공부하는 파이썬** 교재를 기반으로 파이썬의 기초 개념을 정리한 것입니다. 파이썬 설치부터 자료형, 조건문, 반복문, 함수, 클래스 등 다양한 기초 문법을 학습하였습니다.

## **주요 특징**  
- 파이썬 설치 및 기초 문법 학습  
- 자료형, 조건문, 반복문, 함수, 클래스  
- 예외 처리 및 모듈 사용  

## **기술 스택**  
- **Python**
